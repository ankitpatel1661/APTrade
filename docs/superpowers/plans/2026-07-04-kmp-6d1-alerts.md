### Task 1: Shared alerts core

Read Sources/APTradeDomain/PriceAlert.swift + Sources/APTradeApplication/AlertUseCases.swift
first — they are the transcription reference.
1. domain/PriceAlert.kt: sealed AlertCondition — PriceAbove(Money), PriceBelow(Money),
   PercentChange(Double magnitude, fires on EITHER direction: abs(changePercent) >=
   abs(threshold)); summary() strings ≡ Swift ("Price above $X", "Moves N% in a day");
   PriceAlert(id String — use the generateTradeId 128-bit precedent, symbol, condition,
   createdAtEpochSeconds, isTriggered), isMet(quote): pure (>=, <=, abs), triggered()
   copy. application/AlertPorts.kt: AlertStore (load/save List<PriceAlert>) +
   AlertNotifier (suspend notify(alert, quote)).
2. application/AlertUseCases.kt: LoadAlerts, CreatePriceAlert (append+save),
   RemovePriceAlert (filter by id+save), EvaluateAlerts(store, notifier,
   isNotifyEnabled: suspend () -> Boolean) — quotes map in; untriggered alerts matching
   isMet become triggered; persists IF anything changed; notifies per newly-triggered
   ONLY when isNotifyEnabled() (macOS: triggering/persisting happens regardless — the
   gate suppresses only the push; KDoc this). CancellationException-first.
3. Tests (≥10): isMet all three conditions incl. BOTH percent directions + boundary
   equality; evaluate marks+persists; no-save-when-nothing-matched; notify gating both
   ways; already-triggered alerts skipped; create/remove round-trips.
